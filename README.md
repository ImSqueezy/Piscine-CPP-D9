## Compilation

## ex00/
The goal of this exercice is to take a Database as an argument and output the result of the value multiplied
by the exchange rate (according to the date indicated in the Database).

This exercice's program has some rules:
The name must be ./btc and it must take a file as an argument and each line from that must follow the format "date | value", with a valid date of year-month-day, while the btc values must be either integers or floats ranging between 0 and 1000.


Usage:
```bash
./btc [argument]
```
